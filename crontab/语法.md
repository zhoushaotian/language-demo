```bash

    # crontab语法解释
    #
    # +---------------- 分钟 (0 - 59)
    # |  +------------- 小时 (0 - 23)
    # |  |  +---------- 日 (1 - 31)
    # |  |  |  +------- 月 (1 - 12)
    # |  |  |  |  +---- 星期 (0 - 6) (0=Sunday,1=Monday,2=Tuesday,3=Wednesday,4=Thursday,5=Friday,6=Saturday)
    # |  |  |  |  |
    # *  *  *  *  *  command to be executed
    #--------------------------------------------------------------------------
    
    
    
    支持聚类,比如
    1-59 * * * * command
    
    1-59 2,3,4,9-20 * * * command

```
